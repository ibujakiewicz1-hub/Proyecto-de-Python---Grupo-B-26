# Proyecto-de-Python---Grupo-B-26
# Gestión de Estacionamiento  ## Descripción  Este proyecto consiste en un sistema de gestión de estacionamiento desarrollado en Python y ejecutado por consola. Permite registrar el ingreso y egreso de vehículos, controlar los espacios disponibles, calcular el importe a pagar y mostrar estadísticas básicas del estacionamiento.
